How It Works

Automatic Launch Mode

Open a .toe file with TD Launcher Plus and it instantly analyzes the file, detects the required version, and presents a 5-second countdown.

  • Auto-launch: Left undisturbed, the project opens in the detected version
  • Interrupt: Click or press any key to stop the timer and choose manually

Dashboard Mode

Launch the app directly to open the project dashboard. Browse Recent Files and Templates, view project info, and manage your workflow.

  • Smart Discovery: Scans for all installed TD versions
  • Integrated Docs: README.md loads instantly per project
  • First-Run Setup: Guided .toe file association on first launch

Recent Files

TD Launcher Plus keeps track of recently opened projects. Files are added when actually launched — not when simply browsing.

Native Recent Files Discovery

  • Windows: Reads recent files directly from the Windows Registry — no setup needed.
  • macOS: Install the companion TDLauncherPlusUtility.tox to sync recent files from TouchDesigner.

Smart Versioning

TouchDesigner auto-save versions like project.7.toe are displayed as their base name. Multiple versions of the same project appear only once. Backup folder files are shown with a Backup/ prefix.

Templates

Save frequently-used project templates for quick access in a separate tab. A built-in "Default (new project)" entry is always at the top for launching TouchDesigner with its default startup file.

  • Click "Add Templates..." to add .toe files as templates
  • Templates default to the latest installed TD version
  • Reorder with the arrow buttons or Cmd/Ctrl+Up/Down
  • Templates persist between sessions

Project Icons

Enable "Show Icons" to display project thumbnails. Icons are auto-generated by the companion utility TOX, or you can place your own.

Icon Search Order

  1. <projectname>_icon.png — static icon (generated on-demand via Utility TOX)
  2. <projectname>_icon_temp.png — dynamic icon (auto-generated on-save)
  3. icon.png — generic project icon
  4. Any other image in the project folder (most recently modified)
  5. Falls back to the TD Launcher Plus app icon

Companion Utility TOX

A TouchDesigner component (TDLauncherPlusUtility.tox) that integrates with TD Launcher Plus. Add it to your default startup file.

macOS — Required for Recents Windows — Optional (Icons)
  • Auto-generate icons: Snapshots the /perform window on every save
  • Manual icon creation: Create custom icons that take priority over auto snapshots
  • Quick access: Open TD Launcher Plus from within TouchDesigner

Installation

  1. Import TDLauncherPlusUtility.tox into your project
  2. Or better: add it to your default startup .toe file so it's always present — it's lightweight and only does minor file operations on init and save

Installation

Windows

  1. Download the installer from the releases page
  2. Run the installer
  3. Set Windows to open .toe files with TD Launcher Plus by default
  4. Double-clicking .toe files will now launch them with TD Launcher Plus

macOS

  1. Download the .dmg file from the releases page
  2. Open the DMG and drag "TD Launcher Plus" to Applications
  3. Important: The app is not notarized — see the security note below
  4. File association is automatic. Optionally set as default via right-click → Get Info → Open with

macOS Security Note

Because the app is open-source and not signed with an Apple Developer certificate, macOS Gatekeeper will show a warning.

  1. Double-click the app
  2. When the warning appears, click Done (not "Move to Trash")
  3. Open System Settings → Privacy & Security
  4. Click Open Anyway next to the TD Launcher Plus message
  5. You only need to do this once

Alternative: xattr -cr "/Applications/TD Launcher Plus.app"

Settings

  • Max Recent Files: Configure how many recent files to keep (5–200, default 33)
  • Clear Missing Files: Remove all entries whose files no longer exist — cleans Recent Files, Templates, and Windows Registry entries
